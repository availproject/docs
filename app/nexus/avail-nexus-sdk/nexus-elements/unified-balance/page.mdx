# Unified Balance

Display the user's unified token balances across chains with per-token and per-chain breakdown.

<Callout type="info">
**MAIN POINTS TO NOTE:**<br/>
1. This widget reads balances from the Nexus context which is populated via `getUnifiedBalances(true)` in the provider; it does not perform network calls by itself.<br/>
2. The widget filters out zero balances, groups by token, and shows per-chain breakdown with formatted amounts and approximate fiat.<br/>
3. To refresh balances after actions, call `fetchUnifiedBalance()` from the provider; the widget will re-render with updated context.
4. You can find a quickstart example for using `nexus-elements` in [Nexus Elements Quickstart](/nexus/nexus-quickstart/nexus-elements).
</Callout>
