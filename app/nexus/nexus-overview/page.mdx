## Nexus SDK (Headless)

A powerful TypeScript SDK for crosschain operations, such as token bridging, crosschain swaps, and unified balance management across multiple EVM chains.

The core functionality of the Nexus SDK is provided by the `@avail-project/nexus-core` package.

- [**@avail-project/nexus**](https://github.com/availproject/nexus-sdk): Headless SDK for cross-chain operations
    - No React dependencies required
    - Direct Chain Abstraction integration

## Get Started Fast

### `create-liquid-apps` Starter Kit

Get set up with a boilerplate starter kit instantly and build faster using either Nexus SDK or Nexus Elements using `create-liquid-apps`. It helps you scaffold a new application with Nexus SDK using your preferred framework and auth provider.

<Tabs items={['pnpm', 'npm', 'bun']}>

<Tabs.Tab>
```bash filename="Terminal"
pnpx create-liquid-apps
```
</Tabs.Tab>

<Tabs.Tab>
```bash filename="Terminal"
npx create-liquid-apps
```
</Tabs.Tab>

<Tabs.Tab>
```bash filename="Terminal"
bunx create-liquid-apps
```
</Tabs.Tab>

</Tabs>

You will be prompted to choose your preferred framework and auth provider in a UI that looks like this:
<br/>
<p align="center">
<img src="/img/nexus-overview/create-liquid-apps-1.png" alt="create-liquid-apps" width="100%" />
</p>

Alternatively, you can skip the UI and use the CLI with options:
```bash filename="Terminal"
pnpx create-liquid-apps my-app --framework next --widgets nexus-core --auth wagmi-familyconnect
```
