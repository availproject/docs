In this guide you will install `@avail-project/nexus-core` and its peer dependencies into an existing project. By the end, your project will be ready for SDK initialization in the next step.

<Callout type="info" title="Prefer a ready-made starter?">
If you want a fully scaffolded project with wallet connection and the SDK pre-configured, use the [Quickstart](/docs/nexus/nexus-sdk/quickstart) instead. This guide is for adding the Nexus SDK to an **existing** project.
</Callout>

## Prerequisites

Before you begin, make sure you have:

- **A JavaScript or TypeScript project** ‚Äî the SDK works with any framework or runtime (React, Node.js, vanilla JS, etc.)
- **Node.js 18+** or **Bun** with a package manager (npm, pnpm, yarn, or bun)
- **An [EIP-1193](https://eips.ethereum.org/EIPS/eip-1193) compatible wallet provider** to sign transactions (e.g. MetaMask via `window.ethereum`, or a library like [wagmi](https://wagmi.sh))

<Callout type="info" title="Wallet provider">
The SDK needs an EIP-1193 provider to send transactions. In a browser you can use `window.ethereum` directly. Wallet libraries like wagmi or RainbowKit are optional ‚Äî they just make it easier to manage wallet connections in React apps.
</Callout>

### What gets installed

The `@avail-project/nexus-core` package provides:

- **`NexusSDK`** ‚Äî the main class for initializing the SDK and performing cross-chain operations
- **Bridge methods** ‚Äî `bridge()`, `bridgeAndTransfer()`, `bridgeAndExecute()` for moving tokens across chains
- **Swap methods** ‚Äî `swapExactIn()`, `swapExactOut()` for cross-chain token swaps
- **Balance methods** ‚Äî `getBalancesForBridge()` and `getBalancesForSwap()` for multi-chain token balances
- **Event constants** ‚Äî `NEXUS_EVENTS` for tracking transaction progress
- **Type exports** ‚Äî `BridgeResult`, `EthereumProvider`, `TOKEN_CONTRACT_ADDRESSES`, and more

## Step 2 ‚Äî Verify the installation

Confirm the package was installed by importing and instantiating it:

```ts
import { NexusSDK } from "@avail-project/nexus-core";

const sdk = new NexusSDK();
console.log("Nexus SDK ready");
```

If this compiles without errors, the installation is complete.
