Nexus Elements is a shadcn-based component library built on top of the Nexus SDK that lets you drop working cross-chain UI into your project.

The standalone per-feature widgets (unified balance, fast bridge, fast transfer, deposit, swaps, view history) have been consolidated into a single **Nexus Elements element**. One component covers swap, send, and deposit modes.

## Install

Install the Nexus Elements element with the shadcn CLI:

```bash title="npm"
npx shadcn@latest add @nexus-elements/nexus-one
```

The single element brings in its own dependencies and source files, and exposes `swap`, `send`, and `deposit` modes.
