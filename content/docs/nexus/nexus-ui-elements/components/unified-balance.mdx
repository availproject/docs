<Callout type="warning">
  **Deprecated.** The standalone Unified Balance component is deprecated in favor of
  the Nexus Elements element. Use
  [`@nexus-elements/nexus-one`](/docs/nexus/nexus-ui-elements/components/swaps) — it
  renders the cross-chain unified balance inline as part of the swap, send, and deposit
  flows.
</Callout>

## Migrating

Remove the standalone `<UnifiedBalance />` component and install the Nexus Elements element instead:

```bash
npx shadcn@latest add @nexus-elements/nexus-one
```

The unified balance view (per-token, per-chain breakdown across connected chains) is shown automatically inside the element. See [NexusProvider Setup](/docs/nexus/nexus-ui-elements/get-started/nexus-provider-setup) to wire it up.

For programmatic access to balances, use the SDK directly — see [Fetch Bridge Balances](/docs/nexus/nexus-sdk/reference/bridge-methods/fetch-bridge-balances).
