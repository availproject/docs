title: Nexus SDK
description: Full SDK control for building cross-chain applications with Avail Nexus.
---


A powerful TypeScript SDK for crosschain operations, such as token bridging, crosschain swaps, and unified balance management across multiple EVM chains.

The core functionality of the Nexus SDK is provided by the `@avail-project/nexus-core` package.

<br/><br/>
[@avail-project/nexus](): Headless SDK for cross-chain operations
- No React dependencies required
- Direct Chain Abstraction integration
- Create your custom UI & UX flows
