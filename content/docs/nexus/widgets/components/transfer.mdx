## Installation

```bash
npx shadcn@latest add @nexus-elements/nexus-one
```

## Usage

Wrap your app with the Nexus provider first. See [NexusProvider Setup](/docs/nexus/widgets/get-started/nexus-provider-setup) for provider setup.

```tsx showLineNumbers
import { NexusOne } from "@/components/nexus-one/nexus-one";

const USDC_BASE = "0x833589fCD6eDb6E08f4c7C32D4f71b54bdA02913";
const RECIPIENT = "0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045";

export function SendExample({ address }: { address?: `0x${string}` }) {
  return (
    <NexusOne
      connectedAddress={address}
      config={{
        mode: "send",
        prefill: {
          token: USDC_BASE,
          chain: 8453,
          amount: "0.1",
          recipient: RECIPIENT,
        },
      }}
    />
  );
}
```

## Configuration

Send uses only `mode: "send"`. This mode is exact-out: users enter the token amount to send, pay-with sources are selected, and the flow submits through the v2 transfer execution path.

| Prop | Required | Description |
| --- | --- | --- |
| `config.mode` | Yes | Must be `"send"`. |
| `config.prefill.token` | Optional | Preselects the token to send. |
| `config.prefill.chain` | Optional | Preselects the destination chain. |
| `config.prefill.amount` | Optional | Prefills the exact output send amount. |
| `config.prefill.recipient` | Optional | Prefills the recipient address. A recipient is required before users can submit. |
| `config.allowedSourcePairs` | Optional | Restricts the selectable pay-with token/chain pairs. |
| `config.allowedDestinationPairs` | Optional | Restricts the selectable send token/chain pairs. |
| `connectedAddress` | Optional | Wallet address to use. If omitted, the connected wagmi account is used. |
| `embed` | Optional | Defaults to `true`. Set `false` to render as a modal surface. |
| `open`, `onOpenChange`, `defaultOpen` | Optional | Control the modal when `embed={false}`. |
| `onStart`, `onComplete`, `onError`, `onClose`, `onConnectWallet` | Optional | Host app callbacks. Use `onConnectWallet` to route the element CTA to your app-level wallet flow. |
