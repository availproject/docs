# Get Merkle proof for a particular extrinsic

## Request

- `Path`: `/eth/proof/:blockhash?index=`
- `Method type`: `GET`

## Example Request

<Tabs items={['CURL']}>

<Tabs.Tab>
    
```sh
curl "https://turing-bridge-api.avail.so/eth/proof/0x5bc7bd3a4793132007d6d0d9c55dc2ded2fe721a49bd771c1d290e6a3c6ec237?index=5"
```

</Tabs.Tab>
</Tabs>
