# Fetch all tokens

## Request

- `Path`: `/user/get_all_tokens`
- `Method type`: `GET`
- `Headers`: `Authorization: Bearer <token>`

## Params

- `limit` (required): The limit of users to query.

## Example request

```bash
curl -X GET "https://api.example.com/v1//user/get_all_tokens" \
     -H "Authorization: Bearer YOUR_TOKEN"
```
