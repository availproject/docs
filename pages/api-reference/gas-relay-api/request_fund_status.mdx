# Retrieve details about a fund request

## Request

- `Path`: `/user/request_fund_status`
- `Method type`: `GET`
- `Headers`: `Authorization: Bearer <token>`

## Example request

```bash
curl -X GET "https://api.example.com/v1/user/request_fund_status" \
     -H "Authorization: Bearer YOUR_TOKEN"
```
