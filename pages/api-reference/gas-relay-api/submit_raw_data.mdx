# Submit data to avail using raw byte data

## Request

- `Path`: `/user/submit_raw_data`
- `Method type`: `POST`
- `Headers`: `Authorization: Bearer <token>`, `Content-Type: application/json`

## Params

- `token`: Token name based on `token_map` endpoint result.

## Body Parameters

- Payload as raw byte data.

## Example request

```bash
curl -X POST "https://api.example.com/v1/user/submit_raw_data?token=ethereum" \
     -H "Authorization: Bearer YOUR_TOKEN" \
     -H "Content-Type: application/json" \
     -d '011010101010101010100101'
```
